Modules Topics covered in this class.

1. Introduction

An overview of ICS 311 policies and procedures and the material to be covered in this course.

Learn more...

2. Analysis examples

Start analyzing algorithms with insertion and merge sort.

Learn more...

3. Growth of functions

How do we characterize functions with respect to their limiting, or asymptotic, behavior.

Learn more...

4. Abstract data types

Stacks, queues, lists, and trees.

Learn more...

5. Probabilistic Analysis

Learn to analyze all cases based on a distribution of the probability of each case.

Learn more...

6. Hash Tables

We consider the design and performance characteristics of what might be the best data structure ever invented.

Learn more...

7. Divide and conquer

Divide the problem into subproblems, conquer by solving them recursively.

Learn more...

8. Binary Search Trees

Trees in which each vertex is either a leaf or has up to two non-empty descendents.

Learn more...

9. Heapsort

Heaps are a useful data structure with applications to sorting and priority queues.

Learn more...

10. Quicksort

Quicksort, like Mergesort, takes a divide and conquer approach, but on a different basis.

Learn more...

11. Balanced Trees

An introduction to Balanced (2,4 and Red-Black) Trees

Learn more...

12. Graphs

Introducing graphs, breadth first and depth first search, topological sort, and strongly connected components.

Learn more...

13. Greedy Algorithms

Introducing greedy algorithms, the activity selection problem, and Huffman codes.

Learn more...

14. Dynamic Programming

Dynamic programming is a method for finding the "best" approach among a set of alternatives.

Learn more...

15. Amortized analysis

How to average the cost of expensive operations across the entire mix of operations to create a more accurate analysis.

Learn more...
